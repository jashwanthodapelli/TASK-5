# 🛒 SmartCart – Advanced E-Commerce Web Application

SmartCart is a modern, responsive, and feature-rich e-commerce web application built as a Final Capstone Project for Full Stack / Web Development Internship Task-5.

The project demonstrates frontend development, JavaScript logic, responsive design, authentication system, cart functionality, admin dashboard, performance optimization, and cross-browser compatibility using HTML, CSS, JavaScript, and PHP.

# ▶️ How to Run the Project

## Step 1

Download or clone the repository.

```bash
git clone <repository-link>
```

## Step 2

Open the project folder.

## Step 3

Run using local server.

For PHP support:

```bash
php -S localhost:8000
```

## Step 4

Open browser:

```bash
http://localhost:8000
```
